Getting Started

Follow these simple steps to add EZComponents navbars to your project:

Add the script to your HTML:

<script src="https://cdn.jsdelivr.net/gh/Botman64/EZComponents/components/navbar.js"></script>

Features

The EZNavbar component comes with a variety of features:

  • Support for both horizontal and vertical orientations
  • Clickable site icon that can be linked to the home page
  • Support for emoji icons, Font Awesome icons, and image URLs for item icons
  • Active state to highlight the current page/section
  • Event handlers for clicks and hover events
  • Dynamic styling using methods like setProperty and getProperty
  • Show/hide functionality with the setVisible method
// Available configuration options
